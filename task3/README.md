# Логический вывод на онтологии (COVID-19)

## 1. Предметная область
Выбрана область медицины: диагностика вирусных инфекций.
Поставлена задача классификации: определить, можно ли отнести пациента к заболеванию COVID-19 по наблюдаемым симптомам.

## 2. Онтология
В Protégé создана онтология со следующими классами:
Пациент, Симптом, Наблюдение, Заболевание, COVID19 (подкласс Заболевание), ПодтверждённыйДиагноз.

Определены свойства:
hasSymptom, recordsSymptom, madeFor, indicates.

Созданы индивиды пациента, его симптомов и наблюдений.

Онтология сохранена в Turtle: ontology_covid19.ttl

## 3. SWRL-правило
Задача: классифицировать пациента как COVID-19, если наблюдаются три ключевых симптома:
fever, dry_cough, loss_of_smell.

В разделе SWRL создано правило в функциональной нотации (см. covid_diagnosis_rule.ttl).

## 4. Логический вывод
Reasoner: Pellet.
SWRL Engine: OWL+SWRL → Drools.

Шаги:
1. Reasoner → Pellet → Start.
2. SWRL → Run Drools.
3. Проверено, что индивид patient_1 автоматически классифицируется как ПодтверждённыйДиагноз.

Все вычисления корректны.
